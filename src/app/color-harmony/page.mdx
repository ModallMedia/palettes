# Color Harmony API

This API allows you to generate color schemes based on a given color. You can specify the color schemes to generate, as well as the formats to return the colors in.

```python
# Python (using requests library)
import requests

params = {
  'color': '#ff0000',  # Replace with your color value
  'formats': 'hsl,rgb,hex',  # Replace with desired formats
  'schemes': 'analogous,complementary'  # Replace with desired schemes
}

response = requests.get('https://yourapi.domain.com/api/v1/color-harmony', params=params)
data = response.json()
print(data)
````

```php
// PHP (using cURL)
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://yourapi.domain.com/api/v1/color-harmony?color=%23ff0000&formats=hsl,rgb,hex&schemes=analogous,complementary");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
$data = json_decode($response, true);
curl_close($ch);

print_r($data);
```

```ruby
# Ruby (using net/http)
require 'net/http'
require 'uri'
require 'json'

uri = URI('https://yourapi.domain.com/api/v1/color-harmony')
params = {
  'color': '#ff0000',  # Replace with your color value
  'formats': 'hsl,rgb,hex',  # Replace with desired formats
  'schemes': 'analogous,complementary'  # Replace with desired schemes
}
uri.query = URI.encode_www_form(params)

response = Net::HTTP.get_response(uri)
data = JSON.parse(response.body) if response.is_a?(Net::HTTPSuccess)
puts data
```
